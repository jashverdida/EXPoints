Quick Setup (2 Minutes)

Step 1: Open phpMyAdmin

1. Open your browser and go to: http://localhost/phpmyadmin

2. Click on your database: expoints_db

3. Click the SQL tab at the top

Step 2: Run This SQL Script

Copy and paste the following SQL code into the SQL text area:

CREATE TABLE IF NOT EXISTS `user_info` (
  `id` int(11) NOT NULL AUTO_INCREMENT,
  `user_id` int(11) NOT NULL,
  `username` varchar(50) NOT NULL UNIQUE,
  `first_name` varchar(50) DEFAULT NULL,
  `last_name` varchar(50) DEFAULT NULL,
  `bio` text DEFAULT NULL,
  `exp_points` int(11) DEFAULT 0,
  `created_at` timestamp NOT NULL DEFAULT CURRENT_TIMESTAMP,
  PRIMARY KEY (`id`),
  UNIQUE KEY `username` (`username`),
  KEY `user_id` (`user_id`),
  CONSTRAINT `user_info_ibfk_1` FOREIGN KEY (`user_id`) REFERENCES `users` (`id`) ON DELETE CASCADE
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;

4. Click the Go button

How to Register a New User

  1. Fill in: First Name, Last Name, Email, Password, Confirm Password
  2. Click "CONTINUE"
  3. A modal will appear asking for a username
  4. Enter a unique username (3-20 characters, letters/numbers/underscore)
  5. Click "Complete Registration"
  6. You'll be redirected to the dashboard!

Database Structure

users table: id, email, password, role, created_at

user_info table: id, user_id, username, first_name, last_name, bio, exp_points, created_at

The user_id in user_info links to id in users table
